## Description
This project is a **real-time face detection application** built with **Python** and **OpenCV**.  
It uses the **Haar Cascade Classifier** to detect faces from a webcam feed and highlights them with green rectangles.  

---

## Features
- Real-time face detection
- Lightweight and fast
- Works on laptops and desktops
- Open-source and easy to extend
